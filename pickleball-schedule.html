⚙️ Cài Đặt
Kết nối Google Sheets để kết quả được lưu cloud — xem từ bất kỳ thiết bị nào, bất kỳ lúc nào.
Cách thiết lập Google Sheets:
  1. Mở Google Sheets, tạo spreadsheet mới
  2. Vào Extensions → Apps Script
  3. Xóa code cũ, dán code mới (nút bên dưới), nhấn Save
  4. Nhấn Deploy → New deployment
  5. Type: Web app · Execute as: Me · Access: Anyone
  6. Nhấn Deploy, copy URL và dán vào ô bên dưới
